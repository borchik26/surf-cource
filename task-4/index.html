| Что такое оператор if в языке Dart и как его использовать? |

Оператор if в языке Dart используется для выполнения кода в зависимости от условия, которое должно оцениваться как true или false.

Пример кода:

void main() {
  bool raining = true; // Условие
  if (raining) { // Проверяем, идет ли дождь
    print("Возьмите зонт!"); // Если идет дождь, выводим сообщение
  } else {
    print("Можете выйти без зонта."); // Если не идет дождь, другое сообщение
  }
}

Глоссарий:

  • Условие: Это выражение, которое возвращает значение true или false, определяя, будет ли выполнен код внутри блока if.
  • Блок if: Набор инструкций, выполняемых, если условие истинно.
  • Блок else: Набор инструкций, выполняемых, если условие ложно.

| | Что такое оператор if-case в языке Dart и как его использовать? |

Оператор if-case позволяет осуществлять сопоставление образцов для выполнения кода, если значение соответствует шаблону.

Пример кода:

void main() {
  var pair = [1, 2]; // Массив значений
  if (pair case [int x, int y]) { // Сопоставление образца, чтобы проверить, соответствует ли массив шаблону
    print('Координаты: $x, $y'); // Если да, печатаем значения
  } else {
    throw FormatException('Неверные координаты.'); // Иначе вызываем исключение
  }
}

Глоссарий:

  • Сопоставление образцов: Процесс проверки, соответствует ли значение заранее определенному шаблону.
  • Блок if-case: Блок, в котором осуществляется сопоставление образцов.
  • Переменные образца: Переменные, которые можно использовать внутри блока, если сопоставление успешно.

| | Что такое оператор switch в языке Dart и как им пользоваться? |

Оператор switch используется для проверки значения на соответствие одним из нескольких заранее определенных образцов или значений и выполнения кода в зависимости от результата.

Пример кода:

void main() {
  var command = 'OPEN'; // Значение, которое проверяется
  switch (command) {
    case 'CLOSED':
      print('Команда закрыть'); // Если значение 'CLOSED', выполняется этот блок
      break;
    case 'PENDING':
      print('Команда в ожидании'); // Если значение 'PENDING', выполняется этот блок
      break;
    case 'APPROVED':
      print('Команда одобрена'); // Если значение 'APPROVED', выполняется этот блок
      break;
    case 'DENIED':
      print('Команда отклонена'); // Если значение 'DENIED', выполняется этот блок
      break;
    case 'OPEN':
      print('Команда открыть'); // Если значение 'OPEN', выполняется этот блок
      break;
    default:
      print('Команда не распознана'); // Если ни одно из значений не соответствует, выполняется этот блок
  }
}

Глоссарий:

  • Значение: Конкретное значение, которое проверяется оператором switch.
  • Образец (case): Каждое значение или образец, которое проверяется внутри оператора switch.
  • Блок switch: Набор инструкций, который выполняется, если одно из значений соответствует.

| | Как используйте выражения switch в языке Dart? |

Выражение switch в Dart позволяет определить значение на основе сопоставления с конкретным образцом и возвращает это значение.

Пример кода:

void main() {
  var x = 1; // Исходное значение
  var result = switch (x) {
    1 => 'один', // Если x равен 1, возвращаем 'один'
    2 => 'два', // Если x равен 2, возвращаем 'два'
    _ => 'неизвестное значение', // Иначе возвращаем 'неизвестное значение'
  };
  print(result); // Печатаем результат
}

Глоссарий:

  • Выражение switch: Это выражение, которое возвращает значение на основе сопоставления с образцом.
  • Образец: Конкретное значение или шаблон, с которым сопоставляется выражение.
  • Управляющая конструкция: Блок, который определяет логику сопоставления и возврата значения.

| | Что такое проверка полноты (exhaustiveness checking) в Dart и как ее использовать? |

Проверка полноты (exhaustiveness checking) гарантирует, что для любого заданного значения будет найдено соответствие в выражении switch.

Пример кода:

void main() {
  sealed class Shape {} // Определение запечатанного класса
  class Square implements Shape {
    final double length;
    Square(this.length);
  }
  class Circle implements Shape {
    final double radius;
    Circle(this.radius);
  }
  double calculateArea(Shape shape) => switch (shape) {
    Square(length: var l) => l * l, // Вычисление площади квадрата
    Circle(radius: var r) => math.pi * r * r // Вычисление площади круга
  };
}

Глоссарий:

  • Запечатанный класс: Класс, у которого известны все возможные подклассы на этапе компиляции.
  • Проверка полноты: Операция, которая гарантирует, что все возможные значения будут учтены в выражении switch.
  • Подкласс: Класс, который наследуется от другого класса.

| | Как пользоваться охранной конструкцией (guard clause) в Dart? |

Охранная конструкция (guard clause) используется для добавления дополнительных условий в сопоставление образцов.

Пример кода:

void main() {
  var value = 10; // Значение проверки
  switch (value) {
    case var x when x > 5:
      print('Значение больше 5'); // Печатаем, если x больше 5
    break;
    case var x when x <= 5:
      print('Значение меньше или равно 5'); // Печатаем, если x меньше или равно 5
    break;
  }
}

Глоссарий:

  • Охранная конструкция: Условие, добавляемое к образцу для получения дополнительного управления допустимостью выполнения кода.
  • Сопоставление образцов: Процесс проверки значения на соответствие шаблону.
